## Github CLI Extensions

### Install

```
cd gh/gh-[EXTENSION_NAME]
gh extension install .
```

### Usage

```
gh [EXTENSION_NAME]
```

### Upgrade

`gh` is annoying about upgrading local extensions, need to remove it then re-install

```
gh extension remove gh-[EXTENSION_NAME]
# follow install step
```
